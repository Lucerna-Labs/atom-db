# Cognitive observation experiment

## Hypothesis

A bounded activation field can use context and feedback to retrieve relevant
regions of an immutable atom graph more selectively, while leaving the durable
fact set unchanged and keeping every result explainable as a traversed thread.

## Local laws

1. A cue enters through an on-ramp with fixed activation.
2. Context atoms enter with weaker activation.
3. Every ternary bond is a local intersection among its three atom identities.
4. Activation divides across local degree, so high fan-out creates natural
   backpressure instead of unbounded amplification.
5. Only the strongest bounded frontier remains in working memory at each depth.
6. Signals meeting at an atom accumulate and expose an intersection.
7. Successful supervised recall reinforces bonds in its reconstructed thread.
8. Explored bonds outside the successful thread are inhibited.
9. Conductance above the configured threshold is promoted to the fast layer.
10. Observation half-life moves learned conductance back toward its neutral
    prior.

All arithmetic is deterministic integer arithmetic. Equal facts, configuration,
context, feedback history, and cue produce equal recall measurements.

## Spiderweb mapping

- Durable bonds are ground-level local streets.
- Cue and context injection are on-ramps.
- Reinforced conductance forms emergent threads from actual successful flow.
- Shared atoms are intersections where vibrations accumulate.
- Working-memory capacity and degree division propagate backpressure.
- Promoted bonds form the fast directional observation layer.
- Ranked recalled atoms and explanations are off-ramp results.

The graph snapshot preloads adjacency before activation begins. No permanent
thread is authored in advance.

## Falsification gates

The experiment fails if:

1. the same inputs produce different rankings;
2. context cannot change attention on an ambiguous cue;
3. working memory exceeds its configured bound;
4. successful feedback does not strengthen its recall thread;
5. failed feedback does not inhibit explored alternatives;
6. recall changes the number or bytes of durable facts;
7. an explanation cannot identify the cue, context count, target outcome,
   explored bonds, path length, and promoted work;
8. activation grows without a bound under graph fan-out;
9. learned state silently becomes durable truth;
10. the implementation introduces an external dependency.

## Honest boundary

This stage tests association, contextual attention, reinforcement, inhibition,
decay, and explainable recall. It does not yet implement durable cognitive
checkpoints, chunk formation, abstraction, semantic embeddings, natural-language
understanding, or consciousness. Repeated feedback can create a habit that
overrides a weaker context signal; that bias is a measured behavior, not hidden.
